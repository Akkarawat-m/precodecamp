Tag Attribute Usage Comment
h global / event ใช้ในการกำหนดรูปแบบของ Heading
p global / event ใช้ในการกำหนดรูปแบบของ paragraph
div global / event ใช้ในการกำหนดรูปแบบของ Division/session ใน HTML
img alt ระบุ Alternate Text
height ระบุความสูง
size กำหนด size สำหรับ layout ต่าง ๆ
src ระบุแหล่งที่มาข้องภาพ
width ระบุความกว้างของภาพ
a download ดาวน์โหลดไฟล์เมื่อคลิ๊ก
href ระบุ URL เป้าหมายเมื่อคลิ๊ก
media ระบุสื่อ/อุปกรณ์ที่หน้าที่ลิงค์ไว้ออกแบบมา
rel ระบุ Relationship เช่น alternate author bookmark external help license next nofollow noreferrer noopener prev search tag
target ระบุว่าจะเปิดลิงค์ที่ไหน เช่น _blank _parent _self _top framename
ul global / event ใช้กำหนดรูปแบบของ Unorder List
ol reversed เรียงลำดับของลิสท์จากมากไปน้อย
start กำหนดค่าเริ่มต้นของลิสท์
Type รูปแบบของคำหน้าลิสท์ เช่น 1 A a I i
table global / event ใช้กำหนดรูปแบบของตาราง
form accept-charset กำหนด Encode ของ Character
action ระบุว่าจะส่งข้อมูลไปที่ไหนเมื่อกดส่ง
autocomplete เติมข้อความอัตโนมัติ on / off
enctype ระบุว่าข้อมูลจะเข้ารหัสยังไงเมื่อส่ง
method ระุบ HTTP Method GET / POST
name รุบชื่อ form
novalidate ระบุว่าข้อมูลในฟอร์มไม่ต้องตรวจสอบ
rel ระบุความสัมพันธ์ระหว่างแหล่งที่ลิงค์กับเอกสารปัจจุบัน เช่น external help license next nofollow noopener noreferrer opener prev search
target ระบุปลายทางตอนเปิดฟอร์ม เช่น _blank _self _parent _top
button autofocus ให้ปุ่มถูกโฟกัสอัตโนมัติตอนโหลดหน้า
disabled ปิดปุ่ม
form ระบุว่าปุ่มนี้เป็นของฟอร์มไหน
formaction ระบุว่าจะส่งข้อมูลจากฟอร์มไปที่ไหนตอน submit
formenctype ระบุการเข้ารหัสฟอร์มเมื่อ submit
formmethod ระบุว่าจะส่งข้อมูลในฟอร์มยังไงเมื่อ submit เช่น get post
formnovalidate ระบุว่าฟอร์มนี้ไม่ต้อง Validate เมื่อ submit
formtarget ระบุว่าจะแสดงข้อความตอบโต้ที่ไหนเมื่อ submit เช่น _blank _self _parent _top framename
name ระบุชื่อปุ่ม
type ประเภทของปุ่ม button reset submit
value กำหนดค่าเริ่มต้นของปุ่ม
