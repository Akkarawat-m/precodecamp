เปรียบเทียบ ความแตกต่าง
hr & br hr เป็น block Element ส่วน br เป็น inline element
ol & ul ol จะเป็นลิสท์แบบเรียงลำดับ ul จะเป็นลักษณะแบบ bullet ไม่ได้เรียงลำดับ
select & checkbox & radio button select จะเป็น Dropdown Checkbox จะเป็นแบบติ๊กหน้าช่อง radio button จะเป็นปุ่มกลม
div & span div เป็น block element เวลาขึ้น div ใหม่ จะเป็นบรรทัดใหม่ span เป็น inline เนื้อหาจะต่อกันเป็นบรรทัดเดียว
script & style script เป็นการเรียกใช้ script , style เป็นการกำหนดรูปแบบของเอกสาร html
